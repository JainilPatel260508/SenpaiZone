# SenpaiZone: Smart Anime Dashboard

## Description
SenpaiZone is a simple anime web app made using HTML, CSS and JavaScript.  
It fetches anime data from an API and displays it in a simple way so users can search and explore anime.

## Objective
- Learn API usage in JavaScript  
- Practice DOM manipulation  
- Handle user input  
- Use localStorage  

## Motivation
I wanted to build something more than a basic app, so I added features like search, filters, watchlist and suggestions.

## API Used
### Jikan API
- Used to fetch anime data  
- Provides title, rating, genres and images  

Link: https://api.jikan.moe/v4/

## Features
### Search
- Search anime by name  

### Filtering
- Filter by genre  
- Filter by status  

### Sorting
- Sort by rating  

### Watchlist
- Add anime to watchlist  

### Search History
- Stores recent searches  

### Suggestions
- Shows anime based on last search  

### Dark Mode
- Toggle between dark and light mode  

## Technologies Used
